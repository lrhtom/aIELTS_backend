# AI 英语单词短视频生成功能 (AI Vocabulary Video Generator)

这是一个非常有创意且具有巨大潜力的功能！要实现“输入单词 -> 自动生成包含发音、解释、情景故事和画面的短视频”，我们需要构建一条**多模态的自动化流水线 (Multimodal Pipeline)**。

## 核心架构与工作流 (The Pipeline)

整个功能分为四个核心步骤，全自动化串联：

1. **剧本生成 (LLM Scripting)**：
   - 提取单词的音标、词性、精准释义。
   - 利用当前系统已有的 LLM（如 GPT-5 / DeepSeek）创作一个 15~30 秒的极短剧本（比如一个搞笑的情景小故事、或者电影混剪风格的独白）。
   - 同时让 LLM 为这个剧本拆分出 3-4 个分镜，并写出**每个分镜的绘画提示词 (Image Prompts)**。

2. **配音生成 (TTS - Text to Speech)**：
   - 将生成的英文故事和中文释义文本送到 TTS 引擎（如 OpenAI TTS、Azure 语音或免费的 Edge-TTS）。
   - 生成极具表现力的高质量配音音频文件（MP3/WAV）。
   - 提取音频的字级别或句级别的时间戳（以便后续生成对齐的字幕）。

3. **画面生成 (Image/Video Generation)**：
   - 将 LLM 生成的分镜提示词送到 AI 绘图接口（如 DALL-E 3 或 Stable Diffusion）。
   - 获取 3-4 张高质量、风格统一（如 3D 动画、美漫风、赛博朋克）的场景图片。
   - *(可选进阶)*：使用 Runway Gen-2 或 Pika API 将图片变成微动视频，但成本和耗时会激增，早期建议**纯图片+运镜特效**。

4. **视频合成与压制 (FFmpeg / MoviePy)**：
   - 在 Django 后端使用 `moviepy` 或直接调用 `ffmpeg`。
   - 将音频、生成的场景图片（加上缓慢放大平移的 Ken Burns 动效）、背景音乐（BGM）和自动生成的**双语字幕**拼接压制成一个 1080x1920（竖屏竖版，适合手机观看）的 MP4 视频。
   - 返回视频 URL 给前端播放。

## User Review Required

> [!IMPORTANT]
> 这是一个相对重型的功能，我们需要在**体验、成本和开发周期**之间取得平衡。请您审阅以下三个关键的决策点，告诉我您的偏好：

### 1. 画面呈现方案 (成本与速度考量)
- **方案 A（推荐，MVP版本）**：`AI图片 + 动态缩放平移 (Ken Burns 特效)`。出图快（只需调 3-4 次 DALL-E 3 接口），合成极快（20秒内），成本低。完全可以达到抖音/视频号常见的“AI讲故事”视频的效果。
- **方案 B**：`AI生成真实动效视频`。将生成的图片再送到 Runway/Luma 等视频模型。效果极佳，但生成一个短视频可能需要等待 2-3 分钟，且 API 成本非常昂贵。

### 2. 语音生成 (TTS) 服务选择
- **免费方案**：使用 Python 的 `edge-tts`（微软 Edge 浏览器的免费高质量接口），效果其实非常好，完全免费，不需要 API Key。
- **付费高级方案**：接入 OpenAI 的 TTS API（如 `alloy` 声音），发音极其自然且富有感情。

### 3. 字幕对齐方式
- 短视频的精髓在于**动态字幕**（说到哪个词，哪个词就高亮放大）。
- 如果要实现精准的卡点字幕，后端需要用到 `Whisper` 模型去对生成的音频进行时间戳打点。或者使用带有字级别时间戳返回的专业 TTS 服务（如 Azure）。这会稍微增加后端的复杂度和体积（可能需要额外安装一些 Python 库）。

## 阶段性实施计划 (Proposed Phased Approach)

如果你觉得这个大方向没问题，我建议我们将该功能拆分为 **三步走**，逐步上线：

### Phase 1: 概念验证 (剧本+语音)
- 前端：新增“视频实验室”页面，用户输入单词和想要的视频风格（搞笑/悬疑/科幻）。
- 后端：只用大模型写出多模态剧本脚本，并只调用 TTS 生成“英语配音”。此时页面上只播放音频和显示静态文本。

### Phase 2: 静态画面结合与字幕合成 (核心发布)
- 后端：引入 DALL-E 3 生成分镜插图。
- 后端：在服务器上安装 `ffmpeg`，使用 Python 脚本将“配音 + 分镜图片 + 简单字幕”合成为一个粗略的短片 MP4。返回并存入数据库。

### Phase 3: 抖音级别特效优化 (抛光打磨)
- 引入 `Whisper` 打点，生成炫酷的居中放大动态字幕。
- 引入 BGM（背景音乐库）和图片运镜特效。
- 支持分享、下载，并纳入财政体系计算消耗的 AT 币。
